Справка по работе с веб-сервисом на основе Jetty

Роли:

    1. Guest, доступен GET запрос. Логин/Пароль guest/guest
    2. Manager, доступны GET и POST запросы. Логин/Пароль manager/manager

Запросы:

    1. curl -u LOGIN:PASSWORD http://localhost:PORT/prod отправка GET запроса(получение json строки всех позиций).
    2. curl -u LOGIN:PASSWORD --data "product=INFO" http://localhost:PORT/prod отправка POST запроса, где INFO: Название позиции;Название организации;Количество.
